Frequently asked questions

  1. How do I change the page layout?

    Every page has a data-layout attribute in the <body> tag. If you want to change the layout you need to change this attribute.

    • default-sidebar-1

    For demo purposes, the layout is also set with javascript in the scripts/app.js file in the following lines: 

    $('body').attr('data-background', config.background);
$('body').attr('data-navbar', config.navbar);
$('body').attr('data-sidebar', config.sidebar);
$('html').attr('class', 'loading')

    Make sure you remove or comment this lines when you build your files for production.

    Also the index.html and the views inside the pages folders use the following layouts:

    • empty-view-1

    The style for each layout is defined in the styles/layouts folder using the following naming convention:

    For example, for the default-sidebar-1, the .scss file is located at styles/layouts/default-sidebar-1.scss and will get compiled to css/layouts/default-sidebar-1.css. The colors for this layout are defined in the styles/palettes folder.
